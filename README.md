## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/isaiahvh7/low-orbit.git
   cd low-orbit
   ```

2. Install dependencies:

   ```bash
   # Example for npm/yarn projects
   npm install
   # or
   yarn install
   ```

3. Run the project:

   ```bash
   npm start
   # or
   yarn start
   ```
